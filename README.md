# Don't Let Me Go

> *一款关于"在梦境荒岛中学习生存，也在现实生活中学习活下去"的循环成长型荒野求生游戏。*

> 在现实的沉沦中幻想不凡，在梦境的残酷中学会生还。

---

## 🎮 项目简介

《Don't Let Me Go》是一款结合**心理叙事**、**荒野求生**与**现实成长循环**的独立游戏。玩家扮演一个在生活中逐渐沉沦的学生，在梦境与现实的交替中，通过失败、学习和成长，重新找回行动力。

**核心循环：**

```
现实学习 → 梦境实践 → 遭遇失败 → 梦醒反思 → 获得成长 → 再次入梦
```

---

## ✨ 核心特色

- **🔄 梦境与现实双循环**：现实中学习的知识直接影响梦境中的生存能力
- **📚 知识即力量**：不是传统等级系统，而是"理解了什么"才能做什么
- **💀 失败不是终点**：每次梦醒都是一次成长机会，带着新知识再次挑战
- **🧠 心理叙事**：通过生存压力映射现实中的迷茫与成长
- **🏝️ 荒岛求生**：资源管理、工具制作、维持生命状态的生存体验

## 🎯 第一版原型目标

完成一个"**第一天荒岛梦境 + 梦醒 + 第二次入梦**"的基础流程：

- ✅ 控制角色在荒岛上移动、跳跃、交互
- ✅ 拾取基础资源（树枝、石头、草、浆果、淡水）
- ✅ 查看简单背包
- ✅ 制作至少一个基础生存物品
- ✅ 受到饥饿、口渴、体力、精神等状态影响
- ✅ 因失败或完成目标而梦醒
- ✅ 在现实阶段获得新知识/配方
- ✅ 再次进入同一座荒岛，使用新知识推进

## 🚀 快速开始

### 环境要求

- Unreal Engine 5.x
- Visual Studio 2022 / VS Code + C++ 扩展
- Git

### 构建步骤

```bash
# 克隆仓库
git clone https://github.com/yourusername/DontLetMeGo.git
cd DontLetMeGo

# 生成项目文件
# Windows:
GenerateProjectFiles.bat
# macOS/Linux:
./GenerateProjectFiles.sh

# 编译项目
# 在 VS Code 中按 Ctrl+Shift+B 或：
# 在 Visual Studio 中构建项目

# 打开项目
# 双击 .uproject 文件或在 UE 编辑器中打开
```

### 运行游戏

1. 在 Unreal Editor 中打开项目
2. 打开 `Content/Maps/RealityRoom` 关卡
3. 点击 **Play** 开始游戏

## 📖 开发原则

> **先做能玩的，再做漂亮的。**

- ✅ 先做小循环，再做大世界
- ✅ 先做核心系统，再做复杂内容
- ✅ 每完成一个小功能就记录开发日志
- ✅ 每完成一个稳定节点就提交 Git
- ✅ 遇到不会的内容，优先拆小，而不是放弃
